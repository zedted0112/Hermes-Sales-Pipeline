Workflow

Hermes runs a deterministic, script-based pipeline. This is how skills connect end-to-end.

1
Lead Finder
Find local businesses by city + category.
Skill: local-lead-finder
2
Research
Generate structured research JSON and save it.
Skill: lead-research → save_research.py
3
Demo Website
Build a template-driven demo site from research.
Skill: lead-demo-site → build.sh
4
Publish + Pitch
Publish to Pages, then generate outreach pitch with demo link.
Scripts: publish_to_pages.py • Skill: pitch-generator
Data paths
~/.hermes/leads/
  research/{slug}.json     # saved research (input for demo build)
  demos/{slug}/index.html  # generated demo site
  demos/{slug}/meta.json   # template_used + optional demo_url
