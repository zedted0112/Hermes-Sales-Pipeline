## Template sources (future automation)

This doc captures reputable sources where Hermes can fetch **reusable templates** with clear licensing.

### Best sources (open license, easiest automation)

- **Awesome Landing Pages (GitHub)**: `https://github.com/PaulleDemon/awesome-landing-pages`
- **Landkit (GitHub)**: `https://github.com/srmdn/landkit`
- **Tailwind Toolbox Landing Page (GitHub)**: `https://github.com/tailwindtoolbox/Landing-Page`
- **TailAdmin (MIT, GitHub)**: `https://github.com/TailAdmin/tailadmin-free-tailwind-dashboard-template/`

### Marketplace / paid sources (usable, but licensing constraints)

- **BootstrapMade license**: `https://bootstrapmade.com/license`
- **Webflow template licenses**: `https://webflow.com/templates/template-licenses`
- **Envato Market API (ThemeForest downloads for purchased items)**: `https://build.envato.com/api`

### Design sources (good inspiration, not “auto-reuse”)

- **Figma Community licensing** (CC BY for free files; attribution required): `https://help.figma.com/hc/en-us/articles/360042296374-Figma-Community-copyright-and-licensing`

### Explicit warning

- **Dribbble reuse**: Do not reuse shots as templates without direct permission from the original designer.\n  Source: `https://support.dribbble.com/hc/en-us/articles/360056066854-Can-I-use-work-that-I-find-on-Dribbble`

### Notes for later implementation

- Prefer repos where each template is a self-contained folder with `index.html`.\n- Enforce license allowlist by default: MIT / Apache-2.0 / BSD / CC0.\n- Treat GPL/unknown as blocked.\n- Replace non-redistributable demo images with safe stock images during import.
