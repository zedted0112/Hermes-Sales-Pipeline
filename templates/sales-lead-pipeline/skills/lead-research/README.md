# lead-research

Deep-dive public research on one lead → structured JSON for personalized demo sites.

## Pipeline position

```
local-lead-finder → contact-finder → lead-research → lead-demo-site → pitch-generator
```

## Quick start

```
/lead-research Bull's Legacy Gym, Dehradun

Category: Gym
Lead issues: No website, no booking, no social
Phone: +919759816924
Address: 2 Keshav Road, Kanwali Road, Dehradun
```

## Output

`~/.hermes/leads/research/{slug}.json`

Then:

```
/lead-demo-site bulls-legacy-gym
```

## Location

`~/.hermes/skills/lead-research/`
