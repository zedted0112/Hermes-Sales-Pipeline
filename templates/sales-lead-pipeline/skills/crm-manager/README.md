# crm-manager

File-based lead CRM at `~/.hermes/leads/`. Tracks pipeline status locally.

## Statuses

`new` → `contacted` → `replied` → `won` / `lost` / `closed_no_response`

## Quick start

```
/crm-manager save lead Bull's Legacy Gym
/crm-manager status Bulls Legacy
/crm-manager list pipeline
```

## Location

`~/.hermes/skills/crm-manager/`
